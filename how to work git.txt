Как работать с git через консоль

Необходимо установить git на свой ПК, для этого:
1. перейти на официальный сайт github по ссылке https://github.com/git-forwindows/git/releases/download/v2.39.1.windows.1/Git-2.39.1-64-bit.exe и
скачать установщик для операционной системы Windows.
2. Открыть папку созданного ранее проекта и нажав ПКМ, выбрать Git Bash Here:
3. В открывшейся консоли написать: git init, тем самым инициализировать git.
4. Создать учетную запись на https://github.com
5. Необходимо связать репозиторий github с созданной папкой проекта, для этого:
	1. прописать команду:
	git remote add origin {ссылка}
	2. вместо {ссылка} вставляем ссылку на наш проект на github и в конце
	добавляем «.git», пример:
	git remote add origin https://github.com/IvanLevitskij/test.git
	3. далее необходимо авторизоваться (ввести логин и пароль от github).
6. Загрузить результаты своей работы на github через консоль, для этого:
	1. открыть консоль Git Bash.
	2. в консоли ввести:
	git add --all – добавляем все изменения в commit.
	git commit -m "First commit" – создаем новый commit, в который
	будет добавлен наш файл.
	git push origin master – пушим изменения проекта на github.
